## Dependencies
- Node.js
- NPM (should come with Node)

1. Clone the repository.
2. Run `npm install` to install all package dependencies.
3. Run `npm run dev` to start the development server.

- The markup for the site is written using the HTML pre-processor [Pug](https://pugjs.org/api/reference.html).
- The styles for the site are written using the CSS pre-processor [Sass](https://sass-lang.com/guide).
- Final outputs are transpiled, minified, and placed in the `/dist` folder.
